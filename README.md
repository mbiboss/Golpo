## 🎯 What is Golpo?

**Golpo** is a next-generation Bengali story reading platform that combines elegant design, ambient music, and cutting-edge web technologies to create an unforgettable reading experience.

```
🌙 Dark & Sepia Themes     🎵 YouTube Music Integration     📱 Progressive Web App
📖 Page-Based Navigation   🔖 Smart Bookmarks              ⚡ Lightning Fast
```

## 📖 Adding Stories

1. **Create** `.txt` file in `/stories/`
2. **Update** metadata in `script.js`:

```javascript
{
  id: "story-id",
  title: "গল্পের নাম",
  author: "লেখক",
  category: "Romance", // Romance | Horror | Drama | Mystery
  file: "story-id.txt"
}
```

3. **Update** story grid in `index.html`
