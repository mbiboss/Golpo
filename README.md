## 🎯 Getting Started

### 🧩 Prerequisites
- Modern web browser  
- Internet connection (for music streaming)

### ⚙️ Quick Setup
```bash
# Clone the repository
git clone https://github.com/mbiboss/Golpo.git

# Navigate to directory
cd golpo

# Serve locally
python3 -m http.server 5000
